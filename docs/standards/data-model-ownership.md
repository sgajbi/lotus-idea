# Data Model Ownership

- Service: lotus-idea
- Owns only its bounded-context schema.
- Data-mesh posture: proposed producer products and explicit consumer
  dependencies are repo-owned from day one, but certification is `not_certified`
  until implementation, live telemetry, platform source-manifest inclusion, and
  mesh certification pass.
- Source authority: upstream source-owned values must carry provenance,
  freshness, trust metadata, and lineage; `lotus-idea` must not recompute
  official portfolio, performance, risk, suitability, mandate, reporting,
  rendering, archive, or AI-provider facts.
