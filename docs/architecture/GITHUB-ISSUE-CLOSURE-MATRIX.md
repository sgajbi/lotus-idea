## Closure Rules

1. `#225` remains the campaign ledger and is intentionally excluded from the
   actionable closure set.
2. A row can be `locally_fixed` only when implementation, meaningful tests,
   same-pattern scan, docs/wiki/context decision, and PR close intent are
   present on the branch.
3. The PR description should use the `PR Close Intent` column so GitHub closes
   the issue only after merge.
4. The matrix is guarded by `make github-issue-closure-matrix-gate` and by
   `make lint`.
