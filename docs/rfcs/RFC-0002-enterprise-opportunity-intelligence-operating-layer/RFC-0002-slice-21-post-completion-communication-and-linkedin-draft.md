1. No post-completion communication is drafted before RFC-0002 implementation
   proof and final closure evidence pass.
2. The draft or no-post decision references only supported,
   implementation-backed capabilities and lessons.
3. The Lotus thought-leadership ledger, themes, and voice/style guide have been
   checked before drafting.
4. The closure summary records whether a post was drafted, deferred, or
   deliberately skipped.
5. Public communication status is not treated as implementation proof,
   supported-feature proof, or demo-readiness evidence.
